# Test Inputs

This folder contains local-only benchmark inputs for the scholarship pipeline.

## Bundle Layout

Each sample bundle should group one applicant's application set together so shared attachments can be reused across multiple applications in the same bundle.

Planned bundle categories:
- Scholarship Form
- Travel Scholarship Form
- Watched Scholarship Form
- GSS Scholarship Form
- Shared attachments such as University Grades, Residency, Self Economic Declaration, and Nation Membership
- Optional attachments such as Highschool Grades and Nation Registration Confirmation

## Initial Samples

- `applicant_bundle_5_page/` - one full bundle with 5 pages
- `applicant_bundle_2_page/` - one smaller bundle with 2 pages

The test suite uses CSV output for comparison and regression checks.
