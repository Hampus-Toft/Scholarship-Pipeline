# JPEG sample pairs

Put 2-page JPEG samples here and register them in jpeg_pairs.csv.

Required files per sample:
- page 1 image
- page 2 image
- optional expected answers CSV (same 3-column format as expected_answers.csv)

Use relative paths from applicant_bundle_2_page in jpeg_pairs.csv.

## Scenarios

There are now three separate 2-page scenarios:

- Scenario 1: PDF baseline bundle using `applications/scholarship_form/scholarship_form.pdf` with expected values in `expected/expected_answers.csv`
- Scenario 2: JPEG pair `sample_1_page_1.jpg` + `sample_1_page_2.jpg` with expected values in `expected/expected_answers_sample_1.csv`
- Scenario 3: JPEG pair `sample_2_page_1.jpg` + `sample_2_page_2.jpg` with expected values in `expected/expected_answers_sample_2.csv`

Fill each expected file independently because each scenario has different content.
